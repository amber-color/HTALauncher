# HTALauncher

HTALauncherは、Windows用のHTA（HTML Application）ベースのアプリランチャーです。デスクトップ上でアプリのショートカットをグリッドビューまたはリストビューで表示し、簡単に起動できます。

## 機能

- **アプリ管理**: アプリの追加・編集・削除・並べ替えが可能。
- **ビュー切り替え**: グリッドビューとリストビューを切り替え可能。
- **タブグループ**: アプリをグループ化してタブで管理。
- **検索・絞り込み**: 起動時に検索ボックスへ自動フォーカス。文字を入力するとリアルタイムで絞り込み。

---

### ⌨️ キーボード操作

- `↑` `↓` `←` `→` — 選択アイテムを移動（グリッドは2次元、リストは上下）  
- `Enter` — 選択中のアプリを起動  
- 選択中のアイテムは枠線で表示される  

---

### 🖱️ ドラッグ&ドロップ

- グリッドビューでアプリの順序を変更  
- ランチャー画面の余白にファイル/フォルダをドロップしてアプリを追加  
- URLをドロップしてWebショートカットを追加  

---

### 🖱️ 右クリックメニュー

- アイテム上で右クリック → 編集・削除  
- 余白で右クリック → 追加  

---

### 🎨 テーマ

- ダークモード / ライトモード切り替え  

---

### 🧩 アイコン

- カスタム画像  
- Iconify APIから選択可能  

---

### ⚙️ 設定カスタマイズ

- ウィンドウサイズ・グリッド設定・起動時タブなど変更可能  

---

## インストール

1. このリポジトリをクローンまたはダウンロード  
2. `HTALauncher.hta` をダブルクリックして起動  

---

## 使用方法

### ▶ アプリの追加

- 設定画面の「追加」ボタンから追加  
- 余白を右クリック → 「追加」  
- ファイルやフォルダをドロップ  

---

### ▶ アプリの起動

- クリック または `Enter`  

---

### 🔍 検索

- ヘッダーの検索ボックスに入力（起動時フォーカスあり）  

---

## 設定

| 項目 | 説明 |
|------|------|
| テーマ | ダーク / ライト |
| 起動時のタブ | 固定 or 前回復元 |
| 初期表示位置 | ウィンドウ位置 |
| 既定のビュー | グリッド / リスト |
| グリッド：各セル幅 | アイコンサイズ |
| グリッド：列数・行数 | レイアウト |
| 文字サイズ | 小 / 中 / 大 |
| リスト：行高 | 行の高さ |

---

## 設定ファイル

- `config.json`: アプリリストと設定を保存（手動編集可）

---

## システム要件

- Windows（HTA対応）  
- `mshta.exe`（標準搭載）
