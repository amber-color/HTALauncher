# HTALauncher

HTALauncherは、Windows用のHTA（HTML Application）ベースのアプリランチャーです。デスクトップ上でアプリのショートカットをグリッドビューまたはリストビューで表示し、簡単に起動できるようにします。アプリのグループ化、タブ切り替え、テーマ変更などの機能を備えています。

## 機能

- **アプリ管理**: アプリの追加、編集、削除、並べ替えが可能。
- **ビュー切り替え**: グリッドビューとリストビューを切り替え可能。
- **タブグループ**: アプリをグループ化してタブで管理。
- **検索機能**: アプリ名で検索可能。
- **テーマ**: ダークモードとライトモードを切り替え。
- **ホットキー/ホットコーナー**: ショートカットキーで起動。
- **アイコン**: カスタムアイコンやIconify APIからのアイコン選択。
- **設定**: ウィンドウサイズ、グリッド設定などをカスタマイズ。

## インストール

1. このリポジトリをクローンまたはダウンロードします。
2. `HTALauncher.hta`ファイルをダブルクリックして起動します。
3. 初回起動時に設定が自動で読み込まれます。

## 使用方法

1. **アプリの追加**: 設定画面からアプリを追加。名前、パス、アイコン、グループを指定。
2. **起動**: アプリをクリックまたはEnterキーで起動。
3. **検索**: ヘッダーの検索ボックスでアプリを検索。
4. **設定**: 設定ボタンからテーマ、ビュー、ホットキーなどを変更。
5. **ドラッグ&ドロップ**: グリッドビューでアプリの順序を変更。

## 設定ファイル

- `config.json`: アプリリストと設定を保存。手動編集も可能。

## システム要件

- Windows (HTA対応)
- Internet Explorer (Edgeモードで動作)
